# Instrument profile Excel templates

Official starting point for human curation of instrument acoustic/proxy metadata:

- **`instrument_profiles_template.xlsx`** — header-only workbook with required sheets and columns for Phase 1a import.

Copy the template to a local working location before filling in data. Do not commit real curated workbooks unless explicitly approved.

Documentation: [`docs/instrument_profile_importer.md`](../../docs/instrument_profile_importer.md)

Import tool: `tools/import_instrument_profiles_from_excel.py`
